# إضافة خدمة التوصيل

## المهام المطلوبة:
- [ ] إضافة بيانات أسعار التوصيل الافتراضية للمحافظات والمناطق
- [ ] تعديل نموذج الدفع لإضافة خيارات التوصيل (محافظة، منطقة)
- [ ] حساب تكلفة التوصيل تلقائياً وإضافتها للمجموع
- [ ] حفظ تفاصيل التوصيل مع الطلب
- [ ] إضافة إدارة أسعار التوصيل في واجهة البائع
- [ ] اختبار عملية الدفع مع التوصيل

## الخطوات التفصيلية:
1. إنشاء بيانات أسعار التوصيل الافتراضية في localStorage
2. تعديل دالة showCheckout في script.js لإضافة اختيار المحافظة والمنطقة
3. إضافة حساب التكلفة وحساب المجموع الجديد
4. تحديث حفظ الطلب ليشمل تفاصيل التوصيل
5. إضافة قسم إدارة أسعار التوصيل في seller.html وseller.js
6. اختبار التكامل مع عملية الدفع
